<h2 id="iWJf9">方案</h2>
**基本需求的方案：**

1. 通过navigation sdk 实现基本的地图展示、文字搜索和导航规划；
2. AR导航，眼镜端路线箭头（转向信息（左右转、掉头、高速路口等））需要基于navigation sdk内置的<font style="color:rgb(0, 0, 0);">回调拿到每一步导航指令后将导航信息画在眼镜上或者取Routes API 获取路线 ，在眼镜上 UI 里绘制；</font>
3. <font style="color:rgb(0, 0, 0);">语音导航，无直接sdk接口语音导航搜索结果，需要将asr给出来的地址转换成经纬度，可借助搜索api处理，如places api或者place sdk，而后结合navigation sdk进行导航ui展示。</font>
4. **<font style="color:#DF2A3F;">小地图</font>**<font style="color:rgb(0, 0, 0);">，无直接接口，可以通过裁剪、自定义小地图所需展示功能的方式，绘制到眼镜上</font>**<font style="color:rgb(0, 0, 0);">。</font>**
